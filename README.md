# Wild Horizons study API
Backend RESTful API that provides several interesting destinations to visit in on earth.

# Routes (GET)
`/api` - Fetch all destinations
`/api/continent/<continent>` - Fetch destinations from specified continent
`/api/country/<country>` - Fetch destinations from specified country

# Commands

Iniciar API:
``` js
npm start
```

Iniciar API com auto-reload:
``` js
npm run dev
```
